# **2. Installation**

> ### Terminal 1
>
> ```
> npm install
> cd {backend directory path}
> node server.js
> ```
>
> > ### _Example_
> >
> > ```
> > npm install
> > cd ../weather/backend
> > node server.js
> > ```

> ### Terminal 2
>
> ```
> npm install
> cd {frontend directory path}
> npm run dev
> ```
>
> > ### _Example_
> >
> > ```
> > npm install
> > cd ../weather/frontend
> > npm run dev
> > ```
